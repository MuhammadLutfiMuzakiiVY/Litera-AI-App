# Training Workspace

Training code will be added after the first integration slice is stable.

Planned structure:

- `datasets/` for dataset manifests, never raw private student data.
- `preprocessing/` for feature extraction.
- `cnn_classifier/` for 1D-CNN experiments.
- `lstm_dkt/` for Deep Knowledge Tracing experiments.
- `evaluation/` for F1, AUC, calibration, latency, and bias reports.
